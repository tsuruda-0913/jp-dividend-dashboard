# 日本 高配当株 モニタリング ダッシュボード

日本の高配当株の株価をブラウザで一覧でき、**前日終値からの下落率**を色分けで表示します。
さらに日本の株価・経済ニュースがタイムラインで流れます。

## 主な機能

1. **株価一覧** — 現在値・前日比（緑＝上昇 / 赤＝下落）・配当利回り。銘柄リストは Google スプレッドシートから自動取得
2. **下落表（色分け）** — 前日終値からの下落率を色分け
   | 下落率 | 色 |
   |---|---|
   | 5〜10% | 黄色 |
   | 11〜15% | オレンジ |
   | 16〜20% | 濃いオレンジ |
   | 20%超 | 赤 |
3. **手動更新** — 画面上部の「🔄 今すぐ更新」ボタンで最新データを取得（取得結果は1時間キャッシュ）
4. **ニュースタイムライン** — 日本の経済・株価ニュースを新しい順に表示

## 使い方（Mac）

ターミナルでこのフォルダに移動して、次を実行するだけです。

```bash
./run.sh
```

初回は仮想環境の作成とパッケージのインストールが自動で走ります（数分かかります）。
起動するとブラウザで `http://localhost:8501` が開きます。停止は `Ctrl+C`。

> `permission denied` と出る場合は、最初に一度だけ `chmod +x run.sh` を実行してください。

### 手動で動かす場合

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

```

## 監視銘柄の変更（Google スプレッドシート連携）

監視銘柄は **Google スプレッドシートから自動で読み込みます**。シートを編集して「今すぐ更新」を押せば、アプリの銘柄リストに反映されます。

- 対象シート: [スプレッドシートを開く](https://docs.google.com/spreadsheets/d/1rI-X6hIi_FhThMoYGYEDWVkvJHqVMCZQED8_lDweUgw/edit)
- 必要な列: 「コード」「銘柄名」「配当利回り(%)」（列の順番は自由）
- シートは **「リンクを知っている全員が閲覧可」** にしておく必要があります。
- 別のシートを使いたい場合は [stocks.py](stocks.py) の `SHEET_ID` を差し替えてください。

> 名古屋証券取引所の専属銘柄（市場が「名M」など）は Yahoo Finance に東証データが無いため「—」表示になります。

## 補足

- 株価は `yfinance`（Yahoo Finance）から取得します。**15〜20分ほど遅延**することがあります（投資判断は公式の証券会社アプリでご確認ください）。
- ニュースは Yahoo!ニュース等の公開RSSから取得します。配信状況により取得できない場合があります。
- 下落率は「前日終値からの当日の下落率」です（上昇している場合は下落率0%として色は付きません）。
